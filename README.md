# 🚗 UniRide - Carona Universitária

Plataforma desenvolvida em **React** para facilitar a mobilidade acadêmica, conectando motoristas e passageiros da mesma instituição.

## ✨ Funcionalidades
- **Filtros Rápidos:** Atalhos para destinos como Rodoviária e Campus Central.
- **Formas de Pagamento:** Opções de Pix, Dinheiro ou Cartão integradas ao card.
- **Persistência:** Dados salvos no navegador via LocalStorage.
- **Integração:** Botão que abre conversa direta no WhatsApp.

## 🛠️ Tecnologias
- React.js (Vite)
- React Icons
- JavaScript (ES6)
